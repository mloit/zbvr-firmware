# Zion Brock Vintage Radio Firmware
Baseline MicroPython firmware archive for the Zion Brock Vintage Radio

See [Zion's website](https://www.zionbrock.com/radio) for more details

## Installation
Using Thonny, or other tool that can connect to a device loaded with MicroPython, simply upload both the `AMradioSoud.WAV` and `main.py` files onto your device
